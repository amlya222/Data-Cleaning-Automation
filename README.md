# Data-Cleaning-Automation

This project is a frontend for data cleaning automation. Users can upload messy CSV or JSON datasets, preview the parsed data, apply automatic cleanup rules, and download the cleaned result.

## Features

- Drag-and-drop or browse file upload
- CSV and JSON support
- Automatic cleaning rules:
  - trims whitespace
  - removes empty rows
  - removes duplicate rows
  - normalizes numeric values
- Preview of cleaned data
- Download cleaned dataset

## Run locally

Run `app.py` in a terminal after installing all dependency in `requirement.txt`
